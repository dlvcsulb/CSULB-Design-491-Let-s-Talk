Let's Talk

Developer Guidlines

About the Let's Talk

Let's Talk is an action for the Google Home device that users can utilize to screen for depression. Uses the Google API.ai which allows users to interact with their device through voice or text-based conversations powered by AI.

How to install

To use Let's Taln, users do not have to install any applications, however they will need to approve of the application. They may invoke the action by explicitly calling the action name such as “Ok Google, talk to Anti-Depression to start survey”.

What technologies we utilized.

Database -- Google Firebase
Software -- Dialogflow
